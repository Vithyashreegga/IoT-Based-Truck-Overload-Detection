# 🚛 IoT-Based Truck Overload Detection and GPS-Enabled Alert System

An IoT-based smart monitoring system developed to detect truck overloading and uneven cargo distribution in real time using ESP32, Load Cells, GPS, and the Blynk IoT platform.

---

## 📖 Project Overview

Overloaded trucks can cause road accidents, damage highways, reduce vehicle life, and increase fuel consumption. This project provides a real-time monitoring system that continuously measures cargo weight, detects overload and load imbalance, tracks vehicle location using GPS, and sends instant alerts through the Blynk IoT platform.

---

## ✨ Features

- Real-time cargo weight monitoring
- Overload detection
- Load imbalance detection
- GPS-based vehicle tracking
- Blynk IoT cloud monitoring
- Instant push notifications
- LCD display for local status
- Buzzer alert for overload conditions

---

## 🛠️ Hardware Used

- ESP32 Development Board
- 4 × Load Cells
- HX711 Load Cell Amplifier
- NEO-6M GPS Module
- 16×2 I2C LCD Display
- Buzzer
- Power Supply

---

## 💻 Software Used

- Arduino IDE
- Embedded C
- Blynk IoT Platform

---

## ⚙️ Working Principle

1. Load cells measure the cargo weight.
2. ESP32 reads and processes the sensor data.
3. The total weight and weight distribution are calculated.
4. If overload or imbalance is detected:
   - Buzzer is activated
   - LCD displays the warning
   - Notification is sent through Blynk
5. GPS continuously updates the truck's location.
